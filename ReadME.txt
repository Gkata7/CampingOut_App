CampingOut is made with HTML, CSS, Bootstrap, JavaScript, Node.js, Express, MongoDB, Google Maps API, Passport.JS

CampingOut helps camp lovers to find a campsite to rent for x amount of days. It allows you to leave comments/feedback so it can help a user to decide which campsite is the best and to interact with other users.
